# Pokebot
複数のウィンドウを同時に操作して効率的にリセマラするためのツール

## インストール
1. `Pokebot`フォルダーをダウンロードし、どこか安全な場所に保存
2. `Pokebot`内にある`Pokebot.exe`のショートカットをデスクトップに作成
3. そのショートカットをダブルクリックで起動

## 使い方
1. [BlueStacks](https://www.bluestacks.com/) をインストール
2. 同時にインストールされる`BlueStacksマルチインスタンスマネージャー`を起動
3. インスタンスを１つだけ作成し、起動（以降このインスタンスをメインインスタンスと呼称）
4. `Pokebot.exe`を起動
5. `Pokebot`の`Mainwindow`の欄に、メインインスタンスの表示前（例: `BlueStacks App Player`）を入力
6. `Pokebot`の`Resolution`の欄に、メインインスタンスの画面解像度（例: `1080 x 1920`）を入力（縦画面がおすすめ）
7. インスタンスを複数作成し、すべてのインスタンスで以下の設定を行う
8. 設定 > 上位設定（バージョンによっては詳細設定）> ADBを有効化し、接続先のアドレス（例: `120.0.0.1:5555`）を`Pokebot`の`Subwindows`内のリストの右側に入力（`120.0.0.1:`までは書いてあるので、後ろの数字のみ入力）
9.  画面解像度をメインインスタンスの画面解像度と同じにする
10. 表示名を`Pokebot`の`Subwindows`内のリストの左側に入力
11. リスト内の、同期したいインスタンスのチェックボックスをクリックし、チェックを入れる
12. `Launch`ボタンをクリック
13. メインインスタンスで行った操作が他のインスタンスにもコピーされます
